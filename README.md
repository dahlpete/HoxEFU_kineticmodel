# HoxEFU_kineticmodel
This code will simulate the reduction of HoxEFU by NADH.

NADH concentrations and HoxEFU concentrations are specified in the file run_kinetics_v2.py. By default the [HoxEFU] = 50 uM and [NADH] = 10 mM. All electron transfer rate constants describing electron transfer between FeS clusters and the FMN are specified in the .csv file.
